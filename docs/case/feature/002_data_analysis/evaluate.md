# 前端开发能力评分表 — /dashboard/analysis 页面复现评估

> 评估对象完成 `/dashboard/analysis` 数据分析仪表盘页面的复现（基于 Ant Design Pro v6 + Umi Max 4 + antd v6 + ProComponents v3）。
>
> **评估原则**：重点考察功能还原度、数据流正确性和工程素养。具体视觉样式（配色、间距、字号、圆角等）允许自由发挥，只要整体协调、层次清晰即可。评分时功能正确性优先于视觉还原度。
>
> 评分标准：每项 **1–5 分**（1=未实现/严重错误, 3=基本达标, 5=超出预期/优雅实现），总分 **93 分**。
